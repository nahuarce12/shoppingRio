## Changes Made During Migration

### Architectural Improvements

1. **Component Extraction:** Navbar and footer extracted to reusable Blade components (`resources/views/components/nav/main.blade.php`, `resources/views/components/footer/main.blade.php`)

2. **Asset Pipeline:** Migrated from direct `<link>/<script>` tags to Laravel Vite (`@vite` directive)

3. **Routing:** Replaced static `.html` links with Laravel route helpers (`route('name')`)

4. **JavaScript Modularization:** IIFE patterns converted to ES6 modules with named exports

5. **CSS Consolidation:** Single stylesheet with CSS variables instead of scattered inline styles

### Breaking Changes

- **URLs Changed:** `/pages/locales.html` → `/locales`
- **Asset Paths:** `/img/` → `/images/` (Laravel convention)
- **Bootstrap Loading:** CDN removed, now bundled via Vite

### Bug Fixes Applied

- Fixed duplicate Bootstrap loading (was loading via CDN + Vite)
- Corrected navbar overlay conditional rendering (only on home page)
- Removed double-initialization of main.js module
